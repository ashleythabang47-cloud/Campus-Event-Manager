#  Campus Event Manager

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white)
![OOP](https://img.shields.io/badge/Object--Oriented%20Programming-blue?style=for-the-badge)
![Collections](https://img.shields.io/badge/Java%20Collections-Framework-green?style=for-the-badge)
![Status](https://img.shields.io/badge/Status-Completed-success?style=for-the-badge)


A Java console-based application that allows students and staff to manage campus events through a menu-driven interface. The system demonstrates core Object-Oriented Programming principles and provides features for creating, managing, and registering for campus events.

---

##  Features

### Student
- View available events
- Register for events
- Cancel event registrations
- View registration status

### Staff
- Create new events
- Update existing events
- Cancel events
- View all events

### Event Management
- Prevent duplicate event IDs
- Maximum participant limit
- Automatic waitlist using a Queue
- Save events to a file
- Load saved events when the application starts
- Sort and display events

##  How to Run

1. Clone the repository

```bash
git clone https://github.com/ashleythabang47-cloud/Campus-Event-Manager.git
```

2. Open the project in your preferred Java IDE (NetBeans, IntelliJ IDEA, or Eclipse).

3. Compile and run `Main.java`.

4. Follow the on-screen menu to access the Student or Staff functionalities.
